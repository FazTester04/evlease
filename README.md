**Flux** is a streamlined management platform designed to bridge the gap between electric vehicle (EV) providers and eco-conscious drivers. It simplifies the leasing lifecycle, from vehicle discovery to contract finalization.

---

## ✨ Key Features

* **⚡ Smart Inventory:** Real-time tracking of available EV models with battery specs and range data.
* **📋 Digital Leasing:** Seamless digital application process for new lease agreements.
* **📊 Charging Insights:** (Planned) Integration with charging networks to monitor vehicle health.

## 🚀 Getting Started

### Prerequisites
Make sure you have the following installed:
* [e.g., Node.js v18+]
* [e.g., Git]

### Installation

1. **Clone the repository**
   ```bash
   git clone [https://github.com/FazTester04/evlease.git](https://github.com/FazTester04/evlease.git)
   cd evlease


2. **Install dependency**
   ```bash
    composer install
    npm install
    create .env file
    php artisan migrate
    php artisan db:seed
    php artisan storage:link
    php artisan key generate
    
3. **Run the project**
   ```bash
   php artisan serve
   npm run dev
